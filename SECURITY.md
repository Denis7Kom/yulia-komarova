# Security Policy

## Project status

This repository contains a static personal website for Yulia Komarova, published via GitHub Pages.

The project is public for deployment and review purposes only. Public availability of the repository does not grant permission to copy, reuse, redistribute, resell, clone, or modify the website design, source code, text, visual identity, structure, animations, or any related materials.

## Supported version

Only the latest version of the `main` branch is maintained.

| Version | Supported |
| --- | --- |
| `main` | Yes |
| older commits, forks, copies | No |

## Reporting vulnerabilities

If you find a security issue, do not publish exploit details in public issues, pull requests, discussions, comments, or social media.

Report the issue privately to the repository owner through GitHub. If GitHub private vulnerability reporting is available for this repository, use it. Otherwise, contact the repository owner directly and include:

- affected file or URL;
- short description of the issue;
- reproduction steps;
- potential impact;
- suggested fix, if known.

## Out of scope

The following are not considered security vulnerabilities for this static website:

- visual redesign suggestions;
- ordinary browser rendering differences;
- search engine indexing;
- public visibility of files intentionally committed to the repository;
- unauthorized copying of the website by third parties.

Unauthorized copying is handled as an intellectual property and licensing issue, not as a vulnerability report.
